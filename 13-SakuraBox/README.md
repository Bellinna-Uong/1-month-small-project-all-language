A beautiful and interactive Java Swing application that creates a digital wish-making experience. Make wishes and watch them bloom as flowers in a cherry blossom garden!

## Features

- **Wish-to-Flower Transformation**: Type a wish and press Enter to create a new flower
- **Cherry Blossom Garden**: Beautiful sakura tree with blooming flowers
- **Interactive GUI**: User-friendly interface with text input
- **Visual Feedback**: Flowers appear randomly around the tree branches
- **Java Swing UI**: Cross-platform desktop application

## How to Use

1. Compile the Java file:
   ```bash
   javac SakuraBox.java
   ```

2. Run the application:
   ```bash
   java SakuraBox
   ```

3. Type your wish in the text field at the bottom
4. Press Enter to submit your wish
5. Watch as your wish blooms as a beautiful flower in the garden
6. Repeat to create multiple flowers and build your wish garden!

## How It Works

- **Canvas Drawing**: Flowers are drawn on a custom JPanel
- **Wish Management**: Each wish is stored and transformed into a visual flower
- **Random Placement**: Flowers bloom randomly around the cherry blossom tree
- **Aesthetic Design**: Soft pink background with tree and colorful flowers

## Features

- Clean and intuitive interface
- Text input validation (won't accept empty wishes)
- Automatic text field clearing after each wish
- Beautiful sakura tree illustration
- Colorful flower rendering

## Customization

You can modify:
- Tree appearance (color, size, style)

## The Sakura Box Metaphor

In Japanese culture, sakura (cherry blossoms) symbolize:
- Beauty and transience
- New beginnings
- Hope and wishes
- Spring renewal

This application brings that tradition to your computer, allowing you to make wishes and watch them bloom!
