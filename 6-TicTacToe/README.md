# TicTacToe Game

A cute and fun Tic-Tac-Toe game built with Python and Tkinter, featuring a charming Y2K Pink Pixel aesthetic.

## Features

- **Classic Gameplay**: Play the traditional 3x3 grid Tic-Tac-Toe game
- **Y2K Pink Pixel Style**: Beautiful retro-inspired design with pink and purple colors
- **Tkinter GUI**: Interactive graphical user interface
- **Win Detection**: Automatic detection of winners and draw situations
- **Player Feedback**: Visual feedback and messages for game status

## How to Play

1. Run the script: `python main.py`
2. Click on empty squares to make your move (you are X)
3. The computer (O) will make its move after you
4. First player to get 3 in a row wins!
5. Click "New Game" to start another round

## Requirements

- Python 3.x
- Tkinter (usually comes built-in with Python)

## Installation

```bash
python main.py
```
