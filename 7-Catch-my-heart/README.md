# Heart Catcher Game

A fun and engaging arcade-style game where you catch falling hearts on a canvas! Built with HTML5 Canvas and JavaScript.

## Features

- **Canvas-Based Graphics**: Smooth animations and graphics using HTML5 Canvas
- **Interactive Gameplay**: Move your catcher with the mouse to collect hearts
- **Score System**: Track your points as you catch hearts
- **Game Over Detection**: Game ends when you miss too many hearts
- **Beautiful Aesthetics**: Gradient background and smooth animations

## How to Play

1. Open `index.html` in your web browser
2. Move your mouse to control the catcher at the bottom
3. Catch the falling hearts (❤️) to earn points
4. Avoid missing hearts - you have limited lives!
5. Try to get the highest score possible

## Game Controls

- **Mouse Movement**: Move the catcher left and right
- The catcher follows your mouse horizontally

## Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- No external dependencies

## File Structure

- `index.html` - Main HTML file with canvas setup
- `main.js` - Game logic and event handlers
- `style.css` - Game styling and layout

## Customization

You can adjust game difficulty by modifying:
- Heart fall speed
- Number of lives
- Point values
- Catcher size

All parameters are defined in `main.js`.
