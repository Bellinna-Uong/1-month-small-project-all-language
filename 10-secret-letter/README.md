# Secret Letter Scratch Card

An interactive Valentine's Day card with a scratch-off reveal feature! Uncover a hidden love message by scratching away the overlay with your mouse.

## Features

- **Scratch-to-Reveal Mechanic**: Use your mouse to scratch and reveal the hidden message
- **Canvas Technology**: Smooth scratch effect using HTML5 Canvas
- **Secret Messages**: Hidden messages waiting to be discovered
- **One-Click Reveal**: Quick reveal button if you're impatient
- **Romantic Aesthetic**: Hearts and elegant styling for the perfect love note

## How to Use

1. Open `index.html` in your web browser
2. Scratch the canvas with your mouse to reveal the secret message
3. Click "üíù Reveal Everything" button to instantly reveal the full message
4. Read the hidden love note!

## Files

- `index.html` - HTML structure and scratch canvas
- `main.js` - Scratch interaction logic and reveal functionality
- `style.css` - Styling for the card and containers

## How It Works

The scratch card works by:
1. Drawing a scratch-off overlay on a Canvas element
2. Detecting mouse movement over the canvas
3. Erasing pixels from the overlay as you scratch
4. Revealing the message underneath

## Customization

You can easily customize:
- The hidden message (in the HTML or JavaScript)
- Colors and styling (in `style.css`)
- Scratch sensitivity and texture
- Button text and appearance

## Requirements

- Modern web browser with HTML5 Canvas support
- JavaScript enabled
- No external dependencies
